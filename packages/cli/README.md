## Installation

you can install it with npm:

```bash
$ npm install otox -g
```

```bash
$ otox <command> --port=<port>
```

## Example

example file

```js
// runner.js

console.log("hello world!");
console.log("ping pong!");
```

```bash
$ otox node runner.js --port=8080
```

see the result: [localhost:8080](http://localhost:8080)
